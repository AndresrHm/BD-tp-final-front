# Proyecto Final - BD + IA (Detección de Autos en Estacionamientos)

Este proyecto utiliza **YOLOv8** y el dataset **PKLot** para detectar automóviles en imágenes de estacionamientos y almacenar métricas de ocupación en una base de datos.

Este es el front-end del proyecto.
